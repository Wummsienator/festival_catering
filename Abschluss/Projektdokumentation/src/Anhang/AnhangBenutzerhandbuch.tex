\subsection{Benutzerhandbuch}
\label{app:Benutzerhandbuch}

\subsubsection{Willkommen}
\label{app:BenutzerhandbuchWillkommen}
Vielen Dank, dass sie sich für unsere App-Lösung entschieden haben. Dieses Benutzerhandbuch beschreibt die Nutzung der Festival Catering-App,
welche im Rahmen eines Schulprojektes entwickelt wurde. Ziel des Handbuchs ist es, Benutzerinnen eine verständliche Anleitung zur Bedienung der
Anwendung bereitzustellen. Es richtet sich sowohl an Festivalbesucherinnen als auch an das Standpersonal. \\
Die Anwendung ermöglicht es, Bestellungen für Speisen und Getränke digital aufzugeben, zu verwalten und abzuholen. Dadurch sollen Abläufe 
vereinfacht und Wartezeiten reduziert werden.

Im Folgenden werden die wichtigsten Funktionen sowie die einzelnen Schritte zur Nutzung der Anwendung beschrieben.

\subsubsection{Login-Seite}
\label{app:BenutzerhandbuchLoginSeite}

\begin{figure}[htb]
\centering
\includegraphicsKeepAspectRatio{Handbuch_Login_Seite}{0.5}
\caption{Login-Seite}
\end{figure}

Die Login-Seite ist der Einstieg der Anwendung. Hier können sie sich mit dem über die Ticketbuchung bereitgestellten Zugang anmelden. Geben sie die 
Zugangsdaten in die jeweiligen Eingabefelder ein und bestätigen sie mit einem Klick auf \textbf{Login}. Bei einer falschen Eingabe werden sie
dementsprechend informiert.

\subsubsection{Besucher-Seite}
\label{app:BenutzerhandbuchBesucherSeite}

\begin{figure}[htb]
\centering
\includegraphicsKeepAspectRatio{Handbuch_Besucher_Seite}{0.8}
\caption{Besucher-Seite}
\end{figure}

Haben sie sich als Besucher erfolgreich eingeloggt, so befinden sie sich nun auf der Besucher-Seite. Dort stehen ihnen folgende Funktionen zur Verfügung:
\begin{enumerate}
    \item Hier sehen sie ihre Ticketnummer
    \item Ein Stern-Symbol markiert sie als VIP
    \item Hier sehen sie ihr aktuelles Guthaben
    \item Mit dem \textbf{€>}-Button öffnen sie den Dialog zum Aufladen ihres Guthabens
    \item In dieser Tabelle sehen sie ihre offenen Bestellungen mit Informationen zu Standnummer, Wartezeit, Status, Bestellnummer und zum ursprünglichen Auftraggeber
    \item Mit diesem Button gelangen sie zur Bestellungs-Seite, dort können sie weitere Bestellungen aufgeben
    \item Eingabefeld für eine Ticketnummer für die Ticketfreischaltung
    \item Hier können sie eine selektierte Bestellung für ein gewähltes Ticket freischalten
    \item Hier können sie eine selektierte Bestellung stornieren. Ihr Guthaben wird ihnen wieder zugeschrieben
    \item Diesen QR-Code benötigen sie, um bei der Abholung ihr Ticket zu verifizieren
    \item Hier sehen sie alle Benachrichtigungen, die sie über ihr Ticket erhalten
    \item Hiermit können sie sich ausloggen
\end{enumerate}
\clearpage

\subsubsection{Bestellungs-Seite}
\label{app:BenutzerhandbuchBestellungsSeite}

\begin{figure}[htb]
\centering
\includegraphicsKeepAspectRatio{Handbuch_Bestell_Seite}{0.8}
\caption{Bestellungs-Seite}
\end{figure}

Auf der Bestellungs-Seite können sie neue Bestellungen aufgeben. Hier stehen ihnen folgende Funktionen zur Verfügung:
\begin{enumerate}
    \item Über dieses Suchfeld können sie nach einem verfügbaren Stand suchen
    \item In dieser Tabelle stehen ihnen gefundene Stände zur Auswahl bereit
    \item Haben sie einen Stand gewählt, so können sie hier per Doppelklick verfügbare Produkte ihrem Warenkorb hinzufügen
    \item Ihr Warenkorb enthält alle zuvor gewählten Produkte. Per Doppelklick können sie diese wieder aus dem Warenkrob entfernen
    \item Hier können sie Sonderwünsche für die Bestellung vermerken
    \item Haben sie den VIP-Status, so können sie hier ihre bestellung als priorisiert markieren
    \item Hier sehen sie die Gesamtwartezeit und den Gesamtpreis ihrer Bestellung
    \item Hier können sie die Bestellung abbrechen
    \item Mit einem Klick wird hier die Bestellung aufgegeben, sofern ihr verfügbares Guthaben ausreichend ist (sie erhalten entsprechende Hinweise)
\end{enumerate}

\subsubsection{Verkäufer-Seite}
\label{app:BenutzerhandbuchVerkäuferSeite}

\begin{figure}[htb]
\centering
\includegraphicsKeepAspectRatio{Handbuch_Verkäufer_Seite}{0.8}
\caption{Verkäufer-Seite}
\end{figure}

Haben sie sich als Standpersonal angemeldet, so landen sie in dieser Sicht. Hier stehen ihnen folgende Funktionen zur Verfügung:
\begin{enumerate}
    \item Hier sehen sie ihre Standnummer
    \item Hier sehen sie alle aktuellen Bestellungen, die ihrem Stand zugeordnet sind. Sie erhalten Informationen über Zeitstempel, Wartezeit, Status und ob die Bestellung
    priorisiert wurde
    \item Hier können sie den Status einer selektierten Bestellung weiterschalten
    \item Hier können sie eine selektierte Bestellung stornieren
    \item Hier können sie Produkte für den Nachfüllprozess auswählen
    \item Hier können sie eine Menge für den Nachfüllprozess auswählen
    \item Hiermit aktualisieren sie den Bestand des gewählten Produktes mit der gewählten Menge
    \item Hiermit können sie sich ausloggen
\end{enumerate}
